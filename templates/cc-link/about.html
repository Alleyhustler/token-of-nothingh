About

CC-LINK is made by me, cc! I decided to make this template for fun and because I was asked if I'd be OK with people being heavily inspired by my website's design and I was like... not only am I totally fine with it, why don't I make the code easily available for whoever wants it?

Because I designed my website from scratch from the get go, I don't really know what other templates are like... but hopefully this is easy to use, or at least interesting to look at!

Please let me know if you end up using the template, or if you run into any problems with it, or if you have any feedback.
