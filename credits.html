All art on my website is made by myself unless otherwise stated. I will add a list of credits for any external art I end up using.

Graphics

  • I've used Broider for the pixel art borders around certain sections. It's a neat tool for making quilt-like pixel art borders!

Fonts

  • DM Mono by Colophon Foundry, available through Google Fonts. I self-host this font using the Google Webfonts Helper tool by Mario Ranftl.
  • Degheest: FT88 Extended, distributed by Velvetyne Foundry. Velvetyne is probably the coolest open-source type foundry out there and I love their work and attitude.
  • Terminal Grotesque, also distributed by Velvetyne Foundry.

Misc

  • I use the Deploy to neocities Github action to update my website automatically. There is a template for easy installing. Requires learning about source control but 100% worth it if you're already comfortable with it.
  • status.cafe is where I post the micro-updates you see on my homepage. Love that so many people on there have neocities webpages.
  • I use HTML Comment Box for my guestbook. It's a bit of a pain to customise the CSS... but it's possible!
  • I took an extensive look at Chris Coyier's CSS-Tricks snippet on Seamless Responsive Photo Grids to build my own image gallery pages.
